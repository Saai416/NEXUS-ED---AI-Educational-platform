Implementation Plan: Fix Query Retrieval Issue
Problem
The first question "What are the lab facilities in northeastern university?" fails and returns "I don't know", while the second question with explicit instructions "give response from pinecone and neo4j" succeeds.

Root Causes:

Neo4j Cypher query generation fails to map vague terms ("lab facilities") to actual schema labels
Vector search may not find relevant chunks due to terminology mismatch
When both sources fail, the system correctly responds "I don't know"
